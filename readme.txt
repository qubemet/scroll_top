Scroll Top - Qubemet plugin allows users to easily scroll back to the top of the page..

== Description ==
Scroll Top - Qubemet is a plugin from <a href="https://qubemet.com" target="_blank">Qubemet</a> that enables visitors to easily scroll back to the top of the page. Users can control various settings through dynamic options. The Scroll Top plugin offers the following features.

### Features
* Attractive dashboard with all settings.
* Dynamic Option like choose icon, enable/disable, button position, button text and so on.
* Attractive animation.
* Dynamic style like color, background, size.

== Installation ==

1. Click Plugins/Add New from the WordPress admin panel.
1. Search for "Scroll Top", install and active.

-or-

1. Download the .zip package.
1. Unzip into the subdirectory 'qmt-scroll' within your local WordPress plugins directory.
1. Refresh plugin page and activate plugin.
1. Configure plugin using *Scroll Top - Qubemet* link under 'Settings' menu.
